
## 项目定位

阅读Sigma 更偏向本地阅读场景：本地 TXT / EPUB 导入、书架整理、离线搜索、缺失文件检查、替换净化和阅读定位都做了更多优化，适合把书留在自己设备上的使用方式。

## 说明

本仓库为个人维护发布版本，代码改动全程由 AI 编写，本人未手写代码，主要负责提出需求、测试效果和整理发布。README 重点记录和展示这个分支在本地阅读场景下的改动与取向。

## 与 Luoyacheng 版的主要区别

- 更偏本地阅读：主页搜索默认只搜本地书籍，本地书可按书名、作者、标签和分组筛选。
- 更完整的本地书架管理：支持手动检查缺失本地书籍，删除时同步清理章节、标签、分组排序、书签、阅读记录、搜索记录、缓存和封面。
- 更强的阅读定位：新增“全文起效”开关，全文搜索支持替换净化后的内容，命中定位也做了优化。
- 更适合离线使用：强化本地 TXT / EPUB 导入、智能扫描、WebDAV 备份恢复和本地密码加密。
- 更克制的入口：隐藏了一些偏在线的入口，整理了书架多选、本地导入和阅读页操作，减少不必要的打扰。
